# Docker Study
> This is a simple repo to learn the basics in setting up a
> *Web development environment with NginX, MySQL, PHP, and Laravel 5.8 inside a __Docker__ container*.

## Web Stack
- Docker Containers
    - nginx
    - mysql
    - php
    - composer
    - npm
    - artisan
- NginX
    - nginx:stable-alpine
- MySQL 5.7
    - mysql:5.7.22
- PHP 7.4
    - php:7.4-fpm-alpine
- Composer
  - composer:latest
- NPM
  - node:16-alpine
- Laravel 5.8
