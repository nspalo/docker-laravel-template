# Docker Study
> This is a simple repo to learn the basics in setting up a
> *Web development environment with NginX, MySQL, and PHP inside a __Docker__ container*.

## Web Stack
- Docker Containers
    - nginx
    - mysql
    - php
- NginX
    - nginx:stable-alpine
- MySQL 5.7
    - mysql:5.7.22
- PHP 7
    - php:7.2-fpm-alpine
